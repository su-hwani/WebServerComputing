## 📜 서비스 내용

> **DRF(Django Rest Framework) 를 이용한 게시판 커뮤니티**를 만들었습니다.
> 

## 🛠 기술 스택

> Django
HTML, CSS
> 
## 🙋🏻 팀원 및 개발 기간

## 🖥 개발 내용

1. 질문 및 답변 수정에 대한 히스토리 viewing
2. 질문 댓글에 대한 페이지네이션과 추천순, 최신순 정렬 기능
3. 댓글 추천 기능 
4. 댓글 수정 횟수 표시 기능  
5. comment view 분리

## 💡 성장 경험

### 새로운 경험

- Python 을 사용하는 Django 프레임워크에 대해 배우는 기회가 되었습니다.
- Httpie 를 이용하여 웹과 서버간 통신을 구현하였습니다.

### 웹 개발에서 필요한 기능 구현

- 웹 개발에 있어, 필수적으로 필요한 기능들을 구현했습니다.
    - 웹페이지 구현, 회원가입, 페이지네이션, 네비게이션, 정렬 기능, CRUD 구현
